# 改札ダッシュ！駆け込み乗車シミュレーター (Python CUI版) 🏃

本Webアプリケーション（React/TypeScript版）のロジックとゲームプレイを、Python環境のコマンドライン（ターミナル）でも遊べるように移植したプログラムです。

## 特徴
- **標準ライブラリのみで動作**: 特別な外部パッケージ（Pygameなど）を一切インストールすることなく、Python環境さえあれば即座に実行可能です。
- **おなじみの3つの難易度（ダイヤ）を収録**:
  - `昼下がりの各駅停車`: 練習に最適なお昼のゆったりダイヤ
  - `平日朝の通勤特快`: 行列も混雑度も最大級のラッシュアワー
  - `金曜深夜の終電間際`: タイムリミットが非常にシビアな最終電車
- **コンコースの障害物避けミニゲーム**: ターミナル上で左右のレーンを選んで歩行者を回避するアクションパートを搭載。

---

## 🚀 実行方法

1. **Pythonのインストール**
   お使いのPCにPython 3がインストールされていることを確認してください。
   ```bash
   python --version
   ```

2. **ゲームの実行**
   ターミナル（またはコマンドプロンプト）を開き、`game.py` があるディレクトリに移動して、以下のコマンドを実行します。
   ```bash
   python game.py
   ```

---

## 🎮 ゲームの流れ

1. **ダイヤ選択 (Select Level)**
   - 3つの難易度から1つを選択。
2. **改札タッチ (Gate Timing)**
   - 電車が到着し発車するまでのカウントダウンに合わせて、「今だ！」という瞬間に `Enter` キーを押して改札をタッチします。
3. **コンコースダッシュ (Concourse Dash)**
   - 階段へ向かって走る中、上から降ってくる歩行者を `a` (左), `s` (中), `d` (右) のキーで回避します。ぶつかるとタイムロス！
4. **号車選択 (Door Select)**
   - ドアごとの「混雑状況」「行列人数」を見て、並ぶ号車を選択します。
5. **乗車シミュレーション (Boarding Simulation)**
   - ドアが開き、1人ずつ順に乗車していきます。満員になるかドアが閉まる前に自分が乗り込めれば成功！
6. **結果発表 (Result)**
   - 改札タッチから乗車完了までの合計タイムが表示され、タイムに応じたランク（ゴールド・シルバー・ブロンズ）が授与されます。
